Questions‌

1. What is the Internet? How did it evolve?

2. Write three main characteristic features of the Internet that define it.

3. Name the four basic services provided by the Internet. Explain how each one helps the Internet users.

4. What is an electronic mail (e-mail)? Why do users prefer it to paper mail, telephone, and fax services?

5. What is file transfer protocol (FTP)? Write the steps involved in downloading/uploading a file by using FTP service.

6. Explain how FTP and Telnet services ensure that only authorized users can access resources of a remote computer.

7. Differentiate between “downloading” and “uploading” of information.

8. Write three uses of telnet service offered by the Internet.

9. What is an anonymous FTP site? Why is it used?

10. What is a newsgroup? How is it useful?

11. Differentiate between moderated and non-moderated newsgroups.

12. Answer the following questions:

a. How does a user create a news group?

b. How does a new member subscribe to a news group?

c. How does an old member unsubscribe to a news group?

d. How a member posts a new message to a news group?

e. How a member reads a message from a news group?

13. What is Internet surfing? Write one feature of a web browser that makes Internet surfing easier and one feature that makes Internet surfing faster.

14. What is hypertext? How is it useful?

15. Define the following terms with respect to the Internet:

a. HTML

b. HTTP

c. Web server

d. Web client

e. Web browser

f. URL

16. What is a WWW browser? What types of navigation facilities do modern browsers support to help users save time while Internet surfing?

17. What is a Java-enabled browser? How is it better than a conventional graphical browser?

18. Explain how a Java-enabled browser supports the following features:

a. Dynamic content

b. Dynamic data types

c. Dynamic protocols

19. What is an Internet search engine? What are the three major elements of an Internet search engine? Explain the role of each.

20. What are the major types of Internet search engines? Write the features of each type.

21. Name three popular Internet search engines. Write their key features.

22. Write eight uses of the Internet.

23. Write three uses of the Internet in education sector.

24. Write three uses of the Internet by commercial organizations.

25. What is IPV6? Why it has been developed and proposed?

26. Write the main limitations of IPV4. Explain how IPV6 takes care of these limitations.

27. Write the main features of IPV6.

28. What is an Intranet? How is it different from the Internet? How is it similar to the Internet?

29. Write three different ways in which an organization can make use of an Intranet.

30. What is a proxy server? Why is it used? Explain how users within an organization access the Internet through a proxy server.

31. Write three uses of a proxy server.

32. What is a firewall? Explain how it can help in providing better security to an organization.

33. What is Internet telephony? Explain how a call takes place by using Internet telephony.

34. What is a microbrowser? How is it different from a desktop browser? What are the two general approaches that microbrowsers use for accessing web data?

35. What is Internet of Things (IoT)? What was the main idea behind IoT?

36. IoT is the enhanced form of the Internet. Explain.

37. What do “things” refer to in Internet of Things?

38. What are “smart objects”? Explain with three examples.

39. Write three benefits of IoT and explain each one with an IoT application.

40. Write one IoT application for each of the following:

a. Smart home

b. Smart city

c. Smart heathcare

d. Smart transportation

e. Smart energy management

41. Write five types of challenges that technologists are currently facing in successful implementation of IoT applications.
