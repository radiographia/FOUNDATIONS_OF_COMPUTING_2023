Service Layers of Cloud Computing‌

Cloud computing architectures categorize the various types of services into the following three layers:

1. Infrastructure as a Service (IaaS). This layer contains hardware resources such as servers, storage systems, switches, routers, etc., which are pooled together and allocated on-demand to user requests for such resources.

2. Platform as a Service (PaaS). This layer contains tools for software development and testing, known as Integrated Development Information (IDE). Software developers are the primary users of services offered by this layer.

3. Software as a Service (SaaS). This layer contains a set of applications offered as a service to end users. For example, applications like e-mail, chat, webcasting, etc. form part of this layer.
