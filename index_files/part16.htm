First Generation (1942-1955)‌

The early computers of Figure 1.3 and others of their time were manufactured using vacuum tubes as electronic switching device. A vacuum tube [see Figure 1.4(a)] was a fragile glass device using filaments as a source of electronics and could control and amplify electronic signals. It was the only high-speed electronic switching device available in those days. These vacuum tube computers could perform computations in milliseconds and were known as first- generation computers.

Most of the first-generation computers worked on the principle of storing program instructions

along with data in memory of computer (stored program concept) so that they could automatically execute a program without human intervention. Memory of these computers used electromagnetic relays, and users fed all data and instructions into the system using punched cards. Programmers wrote instructions in machine and assembly languages because of lack of high-level programming languages in those days. Since machine and assembly languages are difficult to work with, only a few specialists understood how to program these early computers.
