COMMUNICATION PROTOCOLS‌‌

A protocol is a set of formal operating rules, procedures, or conventions that govern a given process. A communication or network protocol, therefore, describes the rules that govern the transmission of data over communication networks. These rules provide a method for orderly and efficient exchange of data between the sender and receiver, and for proper interpretation of controls and data transmitted as raw bits and bytes. These rules are part of data communication software.
