Information Security



Users can trust a computer system and rely on it, only if it protects and manages its various resources and information against destruction and unauthorized access. Obviously, security requirements of different computer systems depend on the environment in which they have to operate. For example, security requirements of a system that has to operate in a military environment are different from the security requirements of a system that has to operate in an educational environment.

In this chapter, you will learn about the security requirements of computer systems, and the various policies and mechanisms, which modern computer systems use to deal with these requirements.
