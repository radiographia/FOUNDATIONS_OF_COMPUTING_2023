SECURITY POLICIES VERSUS MECHANISMS‌

Security policies of a computer system define its security goals, whereas security mechanisms are the methods to achieve these goals. Designers often distinguish between security policies and security mechanisms because they can decide the system's security policies independent of the available technologies, but they have to keep track of available technologies for deciding the system's security mechanisms. It may be difficult to implement the security policies of a system completely with currently available security mechanisms. However, system designers can later add new security mechanisms to implement all aspects of the security policies of the system.
