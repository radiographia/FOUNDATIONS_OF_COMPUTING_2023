Questions‌

1. Write the four main security goals of modern computer systems.

2. What is “need-to-know” principle in computer security? Explain with an example why a designer should take care of this principle while designing the security component of a computer system.

3. Differentiate between passive and active attacks. Which of the two is more harmful and why?

4. Write four commonly used methods of passive attack.

5. What is a Trojan horse program? Write an example of both a passive type and an active type Trojan horse program.

6. Write necessary precautions for preventing a computer system from virus infection. How can a user cure a computer system that is infected with virus?

7. Write important differences between computer viruses and worms. How do they each reproduce?

8. What are the common types of active attacks associated with message communication in a distributed system?

9. What is a nonce? Write an example of an item that can be used as a nonce.

10. What is cryptography? Write three uses of cryptography in designing security components for computer systems.

11. Write three basic requirements that a good cryptosystem must fulfill.

12. Differentiate among ciphertext-only, known-plaintext, and chosen-plaintext attacks with respect to a cryptosystem.

13. Explain how symmetric and asymmetric cryptosystems work. Write their relative advantages and disadvantages. Which of the two is more suitable for each of the following cases (write reasons for your answer):

a. Where both encryption and decryption of information are performed by a trusted subsystem

b. Where different subjects perform encryption and decryption of information

c. Where bulk data encryption is involved

d. For establishing connection between two communicating entities in a distributed system

e. For exchange of messages between two communicating entities in a distributed system

14. What are the basic approaches for user authentication in computer systems? Explain how a system authenticates a user in each of these approaches.

15. Explain the password-based approach for user login authentication. What are the problems associated with this approach? Write solutions to overcome these problems.

16. Write a protocol based on symmetric cryptosystem for one-way authentication of two communicating entities (when A wants to communicate with B, and B wants to verify the identity of A before allowing A to communicate with it).

17. Write a protocol based in asymmetric cryptosystem for one-way authentication of two communicating entities (when A wants to communicate with B, and B wants to verify the identity of A before allowing A to communicate with it).

18. Write a protocol based on asymmetric cryptosystem for two-way authentication of two communicating entities (when A and B want to communicate, and both want to verify the identity of each other before communicating).

19. Why does a computer system need access control mechanism? Write the meaning of following terms in context of access control in computer systems - (a) Object, (b) Subject, and (c) Protection rules.

20. What is a domain? Write three different ways in which domains can be realized in the design of security component of an operating system and explain how domain switching will be done in each case.

21. What is an access matrix? Explain how the following issues can be handled in a security system that uses access matrix for access control:

a. Deciding the contents of access matrix entries

b. Validating access to objects by subjects

c. Allowing subjects to switch domains in a controlled manner

22. Answer the following questions for an ACL-based security system:

a. When a subject accesses an object, how does the system decide whether to allow the subject to access the object in the requested mode?

b. How does the system grant access right for an object to a subject?

c. The owner of an object wants to share the object with another subject allowing it restricted access rights to the object. How does the system enable this?

d. How does the system protect ACLs against user tampering?

e. How does the system revoke an access right that it gave earlier to a subject?

23. What is a digital signature? Write a method to create a digital signature.

24. Write a protocol based on digital signature for ensuring integrity of a message in a communication system.

25. What is a firewall? How can an organization benefit by using a firewall?

26. Write four commonly used types of firewalls and the functionalities offered by each.

27. Explain how an organization can use firewalls to implement a Virtual Private Network (VPN).

28. Write the types of attacks against which a firewall cannot protect an organization.

29. Write the advantages and limitations of firewalls.

30. What is software piracy? Write three types of attacks that fall under software piracy.

31. Write four mechanisms to protect software piracy and explain how each one helps in protecting software piracy.

32. Write the types of security that a sensitive e-mail message requires.

33. What is Pretty Good Privacy (PGP)? What features does it provide? Explain the functioning of PGP.

34. What is Privacy Enhanced Mail (PEM)? Explain how it adds privacy to Internet mail applications.

35. What is Secure/Multipurpose Internet Mail Extensions (S/MIME)? What features does it provide?

36. What is a spam e-mail? Why are spam e-mails harmful?

37. Why do e-mail spammers send spam e-mails?

38. What is anti-spam software? Write four techniques that anti-spam software use for automatic elimination of unwanted e-mails.

39. Explain the terms “false negatives”, “false positives”, “white-list”, and “black-list” in the context of anti-spam software.

40. What is IPSec? What functionalities does it provide? What are its key features?

41. What is Secure Socket Layer (SSL)? Specify its location in TCP/IP protocol stack.

42. Draw the SSL protocol stack. Describe the roles of the four protocols of this stack.

43. Describe the four phases of SSL Handshake Protocol.

44. What is Transport Layer Security (TLS)? Write the main differences between SSLv3 and TLS.

45. What is Secure Electronic Transaction (SET)? Write the three services that it provides.

46. List the parties involved in an electronic transaction with the role of each.

47. Write the sequence of events involved in an electronic transaction process.

48. What are the security requirements of an electronic transaction process? Explain how SET meets these requirements.

49. Explain the dual signature mechanism of SET. How is it useful in an electronic transaction process?

50. Explain why public-key cryptography is the primary mechanism for secured communication between two entities in an open network (where users freely join and quit).

51. Explain the roles of CA and PKI in management of public keys in an open network (where users freely join and quit).

52. List the management functions that PKI supports.

53. What is steganography? Write a method used for steganography.

54. What is steganalysis? Write a method used for steganalysis.

55. What are the important design principles that designers of security components should use as guidelines for designing secure computer systems? Explain why these design principles are important.

56. What is cyber law? What types of issues does it deal with?

57. What is cyber forensics? Differentiate between computer forensics and network forensics.

58. What is CERT? How is it useful?

59. Differentiate between:

a. Security policies and mechanisms

b. External and internal security

c. Passive and active attacks

d. Symmetric and asymmetric cryptosystems

e. Authentication and access control mechanisms
