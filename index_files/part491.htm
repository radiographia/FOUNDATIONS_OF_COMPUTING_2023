INTERNET SEARCH ENGINES‌

An Internet search engine is an application, which helps users locate websites containing useful information and references to such information on the WWW. To search information on the WWW, a user types the description of the information using the user interface of the search engine. The search engine then searches the requested information on the WWW and returns the results to the user. The results enable the user to quickly locate the requested information from the vast ocean of information available on the Internet.
