Questions‌

1. What is a computer network? How is it useful?

2. Identify the basic elements of a communication system and write the purpose of each.

3. Differentiate among simplex, half-duplex, and full-duplex modes of data transmission.

4. Which mode of data transmission is suitable for communication between a terminal and a computer and why?

5. “A full-duplex line is faster since it avoids delay that occurs in a half-duplex circuit.” Explain.

6. Explain the terms “bandwidth” and “baud”.

7. Differentiate among narrowband, voice-band, and broadband communication channels. Write a practical application of each.

8. What is a wire pair? Write its practical uses.

9. What is a coaxial cable? Write its practical uses.

10. Explain how microwave systems enable communication between two distant stations.

11. Explain how communications satellites enable communication between two far off places. What advantages and limitations are there in using a communications satellite?

12. What is an optical fiber? Explain how data communication takes place using optical fiber. What are its advantages?

13. Differentiate between analog and digital transmission of data. Write their advantages and disadvantages.

14. Explain the terms “modulation” and “demodulation”.

15. Why do communication systems use modulation for signal transmission? Describe three different forms of modulation.

16. What are modems? What purpose do they serve in data communication systems?

17. Describe some common factors that one should consider while selecting a modem.

18. Explain the use of “fall back” feature in the design of modems.

19. Differentiate between internal and external modems. Write their relative advantages and disadvantages.

20. What is a FAX modem? How is it more useful than a conventional modem?

21. Describe error correction and data compression features found in some modems.

22. What is an intelligent modem? Write about “callback” feature found in intelligent modems. \

23. What is a repeater? How is it different from an amplifier?

24. When does a communication system use public telephone line to connect two computers? Explain why such a system requires modems.

25. Explain about the following types of services offered by common carriers:

a. Dial-up line

b. Leased line

c. Integrated Services Digital Network (ISDN)

d. Value Added Network (VAN)

26. What is ISDN? Differentiate between narrowband and broadband ISDN.

27. What is a Value Added Network (VAN)?

28. What is a multiplexer? Explain its function with a diagram.

29. Describe two basic methods of multiplexing. Write uses of both the methods.

30. Write differences between FDM and TDM. Which method is suitable for communication between computers and why?

31. What is a concentrator? Justify its use in a data communication system.

32. What is a Front End Processor (FEP)? Illustrate its use.

33. Describe asynchronous and synchronous modes of data transmission.

34. Write relative advantages and disadvantages of asynchronous and synchronous modes of data transmission.

35. Explain how circuit switching links a sender and a receiver in a communication network. What are its advantages and disadvantages?

36. Explain store-and-forward method of message switching. What are its advantages and disadvantages?

37. What is packet switching? Why is this method suitable for digital data communication between computers?

38. What is routing? Differentiate between source routing and hop-by-hop routing methods. Write their relative advantages and disadvantages .

39. Differentiate between static and dynamic routing methods. Write their relative advantages and disadvantages.

40. What is network topology? Describe four network topologies in common use. Write their relative advantages and disadvantages.

41. What is a hybrid network? Why are they used?

42. What is a LAN? What are its main objectives?

43. Differentiate among PAN, LAN, CAN, MAN, and WAN.

44. What is a communication protocol? What functions do these protocols perform?

45. Why do computer networks need communication protocols?

46. What is a network interface card? Explain its usage in a computer system.

47. Why is layering used in the design of communication networks?

48. Describe the various layers of the OSI model of network architecture with functions of each layer.

49. What is internetworking? What are the main issues in internetworking? Differentiate among a bridge, a router, and a gateway.

50. What is a wireless computing system? How is it useful?

51. Differentiate between fixed wireless systems and mobile wireless systems.

52. What are the major challenges in successful use of wireless computing systems?

53. Write three applications that can benefit from wireless computing systems.

54. What is a wireless LAN? Describe the three primary WLAN standards.

55. What is a multihop wireless network? Why do wireless communication systems need such networks?

56. What is Wireless Application Protocol (WAP)? Why do wireless communication systems need it?

57. Describe the various WAP layers, the protocols at each layer, and their functions.

58. Describe the basic architecture of WAP. Explain the role of WAP gateway.

59. What is a distributed computing system? Write advantages and limitations of a distributed computing system.

60. What is a cloud computing system? Write its advantages to end users.

62. Differentiate between a public cloud system and a private cloud system.
