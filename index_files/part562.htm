What is Software Piracy?‌

Software piracy deals with illegal copying and resale of software. A software pirate is anyone who breaks system security for stealing intellectual property in software on that system. Software industry loses several billion dollars per year due to software piracy, therefore, making it a major concern for software developers and software vendors. The three types of attacks that fall under software piracy are:

1. Illegal copying. In this case, an attacker makes copies of software without the permission of its original developer/vendor and distributes/sells it to others illegally.

2. Malicious reverse engineering. In this case, an attacker recompiles and reverse engineers the software that he/she has purchased in order to use one or more of its modules in some other software, which the attacker is developing.

3. Tampering. In this case, an attacker makes unauthorized modifications to software for personal gains. For example, the attacker may tamper with the code of an e-commerce application so that it provides unauthorized discount or free services, or may tamper with the code of watermarked software in an attempt to remove its watermarking so that he/she can make illegal copies of the software.
