Shortcut Method for Binary to Octal Conversion‌

We use the following steps in this method:

Step 1: Divide the binary digits into groups of three (starting from the right).

Step 2: Convert each group of three binary digits to one octal digit. Since there are only 8 digits (0 to 7) in octal number system (refer to Figure 2.1), 3 bits (23 = 8) are sufficient to represent any octal number in binary. Moreover, since decimal digits 0 to 7 are equal to the octal digits 0 to 7, we use binary to decimal conversion method in this step.
