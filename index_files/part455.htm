Place Where Routing Decisions are Made‌

Based on this attribute, routing algorithms are of three types:

1. Source routing. In this method, the source node of a message/packet selects the entire path for the message/packet (all intermediate nodes through which it will travel) before sending it. It includes this routing information with the message/packet.

2. Hop-by-hop routing. In this method, each node maintains information about the status of all its outgoing channels and adjacent nodes, and selects only the next node for the current message/packet and transmits it to that node. Typically, it makes this decision based on channel availability and readiness of adjacent nodes to receive and relay the current message/packet.

3. Hybrid routing. This method combines the first two methods. It uses source routing method to specify only a few major intermediate nodes of the complete path, and hop-by- hop routing method to decide the subpaths between any two of the specified nodes.
