Flip-flop Input Equations‌

We know that in a sequential circuit, outputs of its combinational circuit serve as inputs to its flip-flops. Moreover, we can express each output of a combinational circuit as a Boolean expression. Hence, inputs to the flip-flops of a sequential circuit can be described by a set of Boolean expressions. These Boolean expressions are called flip-flop input equations of the sequential circuit.
