Microsoft Windows Server (Earlier Known as Windows NT)‌

Microsoft Windows Server is a multi-user, timesharing operating system developed by Microsoft. It has UNIX-like features. Hence, it is suitable for use with powerful workstations, networks, and database servers. Like UNIX/Linux, Windows Server and its subsequent versions have native support for networking and network services. We classify such operating systems as Network Operating System (NOS). Its main features are:

1. Unlike UNIX, its native interface is a GUI. The look and feel of Microsoft Windows Server GUI is similar to that of Microsoft Windows' GUI.

2. It supports multiprogramming and takes advantage of multiprocessing on systems having multiple processors.

3. It has built-in networking and communications features. Hence, it enables configuration of any computer with Microsoft Windows Server as a network client or server.

4. It provides strict system security.

5. It has a rich set of tools for software development and system administration.

6. It can run Microsoft Windows applications and many UNIX applications directly.
