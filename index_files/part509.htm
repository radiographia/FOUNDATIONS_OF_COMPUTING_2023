WHAT IS A MULTIMEDIA COMPUTER SYSTEM?‌

A multimedia computer system is a computer having the capability to integrate two or more types of media (text, graphics, image, audio, and video) for generation, storage, representation, manipulation, and access of multimedia information. Normally, data size of multimedia information is much larger than textual information because graphics, animation, audio, or video data representation in digital form requires much larger number of bits than that required for plain text data representation in digital form. Due to this, multimedia computer systems require:

1. Faster CPU (for quicker processing of large amount of data),

2. Larger storage devices (for storing large data files),

3. Larger main memory (for running programs with large data size),

4. Good graphics terminals (for displaying graphics, animation, and video), and

5. Audio I/O devices (for playing any audio associated with a multimedia application program).

Not all multimedia computer systems need to have all these features. For example, a computer system that does not have the capability to handle audio and video media is also a multimedia computer system because it can handle multiple media (text and graphics). However, a full- fledged multimedia computer system must be capable of handling all types of media mentioned above, and hence, must have all the features.
