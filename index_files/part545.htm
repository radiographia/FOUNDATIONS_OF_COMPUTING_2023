Types of Authentication Needed‌

Normally, computer systems need the following types of authentication:

1. User login authentication. It deals with a computer system verifying the identity of a user at the time of his/her login to the system.

2. One-way authentication of communicating entities. It deals with one of the two communicating entities of a communication system verifying the identity of the other entity.

3. Two-way authentication of communicating entities. It deals with mutual authentication, whereby both communicating entities of a communication system verify each other's identity.

Some commonly used mechanisms to perform these types of authentication are presented here. Note that the authentication protocol paradigms described here illustrate the basic design principles only. Normally, a realistic protocol is a refinement of these basic paradigms and addresses weaker environment assumptions, stronger post-conditions, or both.
