Personal-assistance Package‌

A personal-assistance package allows us to use personal computers to store and retrieve personal information, as well as to plan and manage schedules, contacts, finances, and inventory of important items.

Some commonly supported features found in modern personal-assistance packages are:

1. Calendar

2. To-do-list

3. Address book

4. Investments book

5. Inventory book
