WHAT IS MULTIMEDIA?‌

A medium (plural media) is something that a presenter uses for presentation of information. There are two basic ways to present some information:

1. Unimedium presentation. In this case, the presenter uses a single medium to present the information. For example, a music system is a unimedium device, because it presents information by using sound medium only. Similarly, a book having only text (no diagrams) is also a unimedium device, because it presents information by using text medium only.

2. Multimedia presentation. In this case, the presenter uses more than one medium to present the information. For example, a TV system is a multimedia device, because it presents information by using sound and video media. Similarly, a book having both text and diagrams is also a multimedia device, because it presents information by using text, graphics, and images.

Obviously, multimedia presentation of any information enhances comprehension capability of its user(s), because it involves the use of multiple senses by the user(s). For example, students in a class can understand a lecture better, if the lecturer draws illustrative diagrams on board while delivering the lecture (multimedia presentation), instead of making only voice explanation (unimedium presentation).

Modern computer systems commonly use the following media for storage, access, and transmission of information:

1. Text (alphanumeric characters)

2. Graphics (line drawings and images)

3. Animation (moving images)

4. Audio (sound)

5. Video (Videographed real-life events)

Hence, in computer world, multimedia refers to the use of more than one of these media (such as graphics and text, or animation and sound, or video, sound and text) for information presentation to computer users.
