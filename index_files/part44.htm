FRACTIONAL NUMBERS‌

In binary number system, fractional numbers are formed in the same way as in decimal number system. For example, in decimal number system,

0.235 = (2 x 10 -1) + (3 x 10-2) + (5 x 10 -3)

and

68.53 = (6 x 10 1) + (8 x 100) + (5 x 10 -1) + (3 x 10 -2)

Similarly, in binary number system,

0.101 = (1 x 2 -1) + (0 x 2-2) + (1 x 2-3 )

and

10.01 = (1 x 2 1) + (0 x 20) + (0 x 2-1) + (1 x 2-2)

Hence, binary point serves the same purpose as decimal point. Some positional values in binary number system are given here.
