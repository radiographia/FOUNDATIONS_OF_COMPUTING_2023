Front-End Processors (FEP)‌

A front-end processor is a special device connected to a computer, which handles all communication-processing jobs of the computer (see Figure 12.14). Its primary purpose is to offload communication-processing tasks from the computer, thereby allowing the computer to focus on applications and data processing jobs. It virtually acts like an intelligent terminal or like

a secretary/receptionist to the main computer.
