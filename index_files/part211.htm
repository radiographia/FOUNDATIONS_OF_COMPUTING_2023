Relationship between Hardware and Software‌

The following points explain the relationship between hardware and software:

1. Both hardware and software are necessary for a computer to do useful job. Both are complementary to each other.

2. We can load different software on the same hardware to make a computer perform different types of jobs.

3. Except for upgrades (like increasing main memory and hard disk capacities, or adding speakers, modems, etc.), hardware is normally a one-time expense, whereas software is a continuing expense. We buy new software for running on the same hardware as and when

need arises, or funds become available.
