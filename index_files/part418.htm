MULTIMEDIA DATABASE SYSTEMS‌

Traditional database systems are suitable for storage and management of simple data types such as strings and numbers. Many users now use computers for multimedia applications, which use multimedia data such as text, graphics, animation, audio, and video. To support such applications, researchers designed database technologies for storage and management of multimedia data. We saw that a database consists of a collection of related data, while a database management system is a set of programs to define, create, store, access, manage and query databases. Hence, a multimedia database is a collection of multimedia data items, such as text, graphics, animation, audio, and video, and a multimedia database management system is a set of

programs to define, create, store, access, manage, and query multimedia databases.
