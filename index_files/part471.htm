Wireless Technologies and Standards‌

Over the years, researchers have introduced several wireless technologies. Out of these, some popular ones are described here.


1G, 2G, 3G, 4G and 5G Technologies

The era of mobile wireless communication started in 1980. Mobile phones (also known as cell phones because they communicate through cellular networks consisting of cells) and other hand- held mobile communication devices use mobile wireless communication technologies. These technologies have undergone considerable changes since 1980 and a new set of mobile wireless communication technologies has been released roughly every ten years. It has been a practice to categorize a new set of mobile wireless communication technologies as a “Generation” and represent it as “G”. Hence, the first set of such technologies is referred to as 1G (first generation), the second set is referred to as 2G (second generation), and so on. Till date, five such sets of technologies are there, leading to 5G, which is yet to come.

Every new generation made mobile wireless communication faster than before and had improved or new features than earlier generation technologies. To take advantage of the advanced features of a new generation, consumers need to obtain new mobile devices (like a new phone) that supports the new generation, and service providers need to make expensive upgrades to their cellular networks. We describe here the five generations of mobile wireless communication technologies.

1. 1G. Mobile phones were first introduced in early 1980s based on 1G technologies. They could be used for only voice services and had maximum communication speed of 2.4 Kbps. They used analog signals for voice transmission. The technology standard for 1G is called Advanced Mobile Phone System (AMPS). 1G was characterized by poor voice quality, poor battery life, large phone size, poor handoff reliability leading to frequent call drop, and poor security.

2. 2G. Mobile phones based on 2G technologies were introduced in 1991. They could be used for text, picture and multimedia message transmission (SMS and MMS), in addition to voice services. They had maximum data communication speed of 64 Kbps. They used digital signals for both data and voice transmission. The technology standards for 2G are called Global System for Mobile (GSM) and Code Division Multiple Access (CDMA). 2G enabled use of mobile phones for SMS, MMS, conference calls, call hold, and billing based services. As compared to 1G, 2G was characterized by better voice quality, better data handling capability, smaller phone size, better handoff reliability leading to fewer call drop, and better security. 2G still had the limitation of handling complex data such as videos. Few years later, 2.5G was introduced to provide higher data communication speed. It offered maximum data communication speed of 384 Kbps.

3. 3G. Mobile phones based on 3G technologies were introduced in 2000. In addition to voice services and transmission of text, pictures and multimedia data, they could also be

used for video data transmission. 3G mobile phones used digital signals and Wide Band Wireless Network. They had data communication speed in the range of 144 Kbps to 2 Mbps, depending on whether the mobile device during its use is stationary (like user is sitting at a place), moving slowly (like user is walking), or moving fast (like user is moving in a car or train). Hence, data communication speed of at least 144 Kbps was one of the requirements to qualify for 3G. The technology standards for 3G are called Code Division Multiple Access 2000 (CDMA 2000) in America, Universal Mobile Telecommunication System (UMTS) in Europe, and Enhanced Data Rate for GSM Evolution (EDGE). 3G enabled use of mobile phones for “true” multimedia messaging including text, images, voice and video data, video chatting, video conferencing, mobile TV, video games, and web-based applications like e-mail and mobile e-commerce. As compared to 2G, 3G was characterized by better voice quality, better data handling capability, better handoff reliability leading to fewer call drop, global roaming facility, and better security features. People started calling mobile phones having such advanced capabilities as “smart phones”.

4. 4G. Mobile phones based on 4G technologies were introduced in 2010. The aim of 4G was to offer much higher speed of voice and multimedia data communication, better quality of service, improved security features, internet over IP, and all these at much lower cost. 4G offers data communication speed in the range of 10 Mbps to 1 Gbps. The maximum speed of a 4G network when the device is moving fast (like user is moving in a car or train) is 100 Mbps and in case of stationary or slow moving devices (like user is sitting at a place or walking) is 1 Gbps. The technology standard for 4G is called Long Term Evolution (LTE). 4G networks are based on all Internet Protocol (all-IP) packet- switching technology. Earlier generations used only circuit-switching technology (1G and 2G) or a combination of circuit-switching and packet-switching technologies (3G). Two key technologies used in LTE are MIMO (Multiple Input Multiple Output) and OFDM ( Orthogonal Frequency Division Multiplexing). 4G enabled use of mobile phones for mobile web access with sophisticated graphical user interfaces, high definition quality mobile TV (HDTV), video conferencing with mobile devices, IP telephony, high-end mobile gaming services, and mobile cloud computing.

5. 5G. Mobile phones based on 5G technologies are expected to be introduced soon. 5G promises to offer significantly faster data communication speed, higher connectivity density with many more mobile devices in use (approximately 100 billion devices), much wider overall wireless coverage, improved security, and many newer types of mobile applications (like driverless cars, IoT applications, etc.) as compared to previous generations. 5G is being designed to offer maximum data communication speed of 35.46 Gbps, which is 35 times faster than 4G. The technology standard for 5G is called Wireless World Wide Web (WWWW). Two key technologies used in WWWW are Massive MIMO and Millimeter Wave Mobile Communications. 5G will enable use of mobile phones for mobile web access with sophisticated graphical user interfaces, high definition quality mobile TV (HDTV), video conferencing with mobile devices, IP telephony, high-end mobile gaming services, mobile cloud computing, interactive multimedia applications, mobile applications based on device-to-device communication (IoT applications), etc., with far better experience than before.

Wireless LAN (WLAN)

A Wireless Local Area Network (WLAN) is a LAN that enables wireless communication among the equipment, which are part of the LAN. It may be connected further to a more extensive fixed network such as a LAN, WAN, or the Internet. Wireless interface card that has an antenna is the primary component of a wireless LAN. This card is connected to a mobile unit as well as to the fixed network. A wireless LAN has limited range and usually operates in local environments such as a building, hallway, park, or office complex. Its main advantage is that it supports flexibility and mobility, but it has low communication bandwidth. IEEE 802.11 is a popular WLAN standard. Wi-Fi is the embedded technology of WLAN based on the IEEE 802.11 standard. Although it was originally developed for supporting mobile computing devices (such as laptops) in LANs, it is now increasingly used for many other services, including Internet and VoIP phone access, gaming, and basic connectivity of consumer electronics products such as televisions, digital cameras, and DVD players.

The following three are the primary WLAN standards:

1. IEEE 802.11. The IEEE 802.11 standard supports 1 Mbps data rate and several choices of physical medium such as spread spectrum and infrared. It also supports prioritized access to the medium. An additional feature of this standard is battery conservation for inactive or idle wireless users. Many universities and companies are encouraging the use of IEEE 802.11-based LANs for accessing campus computing systems and the Internet. Enhancement of this standard to support 11 Mbps data rate is underway.

2. HiperLAN. The HiperLAN standard supports 23.5 Mbps channel rates. It, however, allows the use of only spread spectrum physical medium. Like IEEE 802.11, it also supports prioritized access to the medium.

3. HiperLAN2. HiperLAN2 is a WLAN standard, which is standardized by ETSI (European Telecommunications Standard Institute). An exciting feature of this standard is that it provides for use of connections that offer different quality of service for different applications. It uses time-division multiplexing of unicast, multicast, and broadcast connections. Many major players in WLAN area have formed HiperLAN2 Global Forum to advance and complement the ETSI standardization process.


WiMAX

WiMAX stands for Worldwide Interoperability for Microwave Access. It is a technology based on the IEEE 802.16 for providing wireless data communication in WMANs (Wireless Metropolitan-Area Networks). It supports both point-to-point link and full mobile cellular type access. It is useful for providing communication facility in those areas where we do not have copper network or cable TV network (often referred to as last mile connectivity). It has the added advantage of interoperability and economies of scale due to international standardization. It currently operates in the frequency band of 3.3 to 3.4 GHz.


Wireless Local Loops (WLLs)

Wireless Local Loops (WLLs) technology supports fixed wireless systems. It provides several MHz of bandwidth, which can offer high-speed Internet access and data transfer service, in addition to basic telephone service. These loops allow long-distance carriers to bypass the existing wired local loops owned by local telephone carriers (which may result in the saving of tremendous access charges). Due to this reason, many long-distance companies in advanced countries (like the U.S.) are looking to build their own WLLs to avoid paying access charges. Moreover, in developing nations where laying of thousands of miles of copper cable is impractical, WLLs can be used to provide telephone and low-speed data transfer services.

Among many choices of technologies for WLL environments, cellular and microcellular systems can be used in the 900, 1800, and 1900 MHz ranges with 9.6 Kbps for 10 to 100 users within a few kilometers range. Network designers can deploy wireless LANs to support WLL for users in smaller areas, but with higher bandwidth requirements. Specially designed systems for fixed wireless access, such as Local Multipoint Distribution Systems (LMDS), can provide high bandwidth (tens of Mbps) in large areas for large number of users, but require a direct line of sight.


Radio-router Technology

Radio-router technology uses a radio-transmission framework for packet-based, broadband, IP wireless communications. It enables building of a radio-router network on top of an existing IP infrastructure, rather than from the ground up like a 4G network. Since IP network technology is already well developed and inexpensive, radio-router systems are relatively easy, quick, and economical to implement.

It uses OFDM (Orthogonal Frequency Division Multiplexing) technology, which divides a single channel into several sub-channels, each at a different frequency. This boosts bandwidth by letting a system carry several transmissions at the same time. Radio-router systems offer a maximum throughput of 1.5 Mbps. OFDM, unlike traditional FDM, uses signal modulation and

demodulation techniques, as well as orthogonal placement of adjacent channels, to minimize interference.


Multihop Wireless Networks

Most wireless network environments use radio channel to connect mobile equipment to a base station in a single hop. The base station itself is part of a wired infrastructure. Consider an environment where there is no wired infrastructure, yet the communication system must maintain connectivity among users who can move around arbitrarily and can at times not be in direct transmission range of each other.Multihop wireless networks, also known as instant infrastructure systems, provide wireless communications capability in such environments.

Instant infrastructure systems play an important role in wireless communication because due to transmitted power constraints, not all radios are within the range of each other. Hence, these systems often need to relay packets from one radio to another to deliver the packets to their intended destinations. These systems, however, are more complex to design and manage than traditional single-hop, centrally controlled systems. This is because they need to have complete support for multimedia, multihopping, and mobility features.


Wireless Application Protocol (WAP)

We saw in the discussion above that many different wireless access technologies exist that are not interoperable. Designing and building networks and applications for each technology would be a difficult job for developers. With the proliferation of the Internet and wireless communications devices, Internet access using mobile devices became important for anywhere, anytime information access capability. However, redesigning of all web sites to support downloading of information by mobile users was certainly not feasible. To address these issues, researchers started working towards the development of a technology that could allow development of applications that are independent of the underlying wireless access technology and that could enable access of information from existing web sites using mobile devices. The result of this was wireless application protocol (WAP).

WAP Forum, a consortium of leading wireless companies, developed WAP specifications. WAP supports interoperability of different wireless networks, devices, and applications using a common set of application and network protocols. In fact, WAP is a stack of protocols, which defines the type and structure of messages that two devices have to use for communicating with each other in a wireless environment. WAP protocol stack contains a set of six layers, which are application layer, session layer, transaction layer, security layer, transport layer, and wireless layer. Figure 12.25 presents a summary of the various WAP layers, the protocols at each layer, and their functions. WAP protocols reduce the problems associated with the use of the Internet protocols for wireless data transfer. They do this by reducing communication data volume by eliminating unnecessary data transfers and using binary code. For instance, WAP has introduced a new Markup Language called the Wireless Markup Language (WML), which is a version of HTML optimized for mobile devices.

Figure 12.26 shows the basic architecture of WAP, which is similar to web architecture, but it uses a WAP gateway for HTTP to WAP and WAP to HTTP translations. That is, the WAP gateway acts as a proxy server to a mobile client and translates requests from WAP protocol

stacks to protocol stacks of the information server. The WAP gateway has encoders to translate the content coming from the server into compact formats to reduce the size of data over the wireless network. Usually, the WAP gateway resides in the company or organization, which provides carrier services to its subscribers. This architecture allows mobile users to access a wide variety of contents and applications. It also allows application developers to build content services and applications that can run on a variety of mobile devices.
