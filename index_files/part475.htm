Grid versus Cloud‌‌

Both cloud-computing and grid-computing paradigms take utility model and offer computing resources on-demand basis. They both charge users of resources based on the amount and types of resources used by a user. However, grids typically process batch jobs with a defined start-to- end point, while cloud services can be continuous. Moreover, cloud computing expands the types of resources to include soft resources such as programming tools, database, and web services. It also expands the applicability of the services to web and enterprise applications.
