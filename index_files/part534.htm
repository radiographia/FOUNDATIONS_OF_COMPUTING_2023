SECURITY GOALS‌

Irrespective of their operating environment, most modern computer systems have the following security goals:

1. Secrecy. The system must allow only authorized users to access the information within the system.

2. Privacy. The system must prevent misuse of information by ensuring that a user uses a piece of information only for the purpose for which the system gave it to him/her/it.

3. Authenticity. A user must be able to verify the authenticity of any received data in the sense that the data was indeed received from its expected sender and not from any other source.

4. Integrity. The system must protect the information within it against accidental destruction or intentional corruption by an unauthorized user.
