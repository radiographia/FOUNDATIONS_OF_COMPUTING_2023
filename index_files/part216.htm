ACQUIRING SOFTWARE‌‌

Earlier, application and system software were included in the purchase price of a computer. Today, software is usually not included in the purchase price of a computer. For most computer manufacturers, the purchase price of a computer includes only its hardware and a minimum of system software. A customer has to pay extra charges for additional system software and application software that he/she wants to purchase.

We can obtain the desired software in one or more ways described here. The description also provides the relative advantages and limitations of each way of obtaining software.
