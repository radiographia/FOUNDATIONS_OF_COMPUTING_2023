What is Authentication?‌

Authentication is the process by which a system verifies the identity of a user (person or program) before it permits the user to access a requested resource. That is, a system's authentication mechanism protects the system (or some resource of the system) from an unauthorized user by verifying the user's identity.

Authentication involves identification and verification. Identification is the process of claiming a certain identity by a user, while verification is the process of verifying the identity claimed by the user. Hence, correctness of an authentication mechanism relies on the verification procedure that

it employs.
