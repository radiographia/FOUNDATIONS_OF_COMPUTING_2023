‌


This glossary presents definitions of some commonly used terms in computing, with which readers of this book should be familiar.


1, 2, 3, 4, 5

1G technology First-generation wireless-networking technologies. 1G technology was analog and could be used for only voice services.

2G technology Second-generation wireless-networking technologies. 2G technology was digital, circuit-based, narrowband, and suitable for voice and limited data communications.

3D printer A printer that prints (creates) three-dimensional objects.

3G technology Third-generation wireless-networking technologies. 3G technology offers data rates high enough for mobile users to work with multimedia web content, videoconferencing, and e-commerce.

4G technology Fourth-generation wireless-networking technologies. 4G technology offers data communication speed in the range of 10 Mbps to 1 Gbps. As compared to 3G, 4G offers better quality of service, improved security features, internet over IP, and all these at much lower cost.

5G technology Fifth-generation wireless-networking technologies. 5G technology promises to offer significantly faster data communication speed, higher connectivity density with many more mobile devices in use, much wider overall wireless coverage, improved security, and many newer types of mobile applications as compared to previous generations wireless-networking technologies. 5G offers 35 times faster data communication speed than 4G.


A

Abacus The earliest device that qualifies as a digital computer. It permits the user to represent numbers by the position of beads on a rack. Simple addition and subtraction can be carried out rapidly and efficiently by positioning the beads appropriately.

Abstraction A key element of object-oriented paradigm, which means that an object's characteristics are broken into manageable chunks as defined and documented in its class description.

Access control Deals with policies and mechanisms to prohibit a user/process from accessing any resource/information that he/she/it is not authorized to access.

Access time Time between the instant data is requested from a storage device and the instant delivery begins.

Accumulator register (A) A register that holds initial data to be operated upon, intermediate results, and final result of processing.

Active attack An attack to a computer system that interferes with normal functioning of the system, often causing damaging effects.

Active database A feature that makes a database capable of monitoring and reacting to specific circumstances of relevance to an application.

Activity ratio Ratio of total number of records in transaction file and in master file.

Ada A multi-purpose programming language developed for and used primarily by the U.S. Department of Defense in military applications.

Adapter card See Add-on board.

Adder A logic circuit capable of calculating the sum of two or more quantities.

Additive Manufacturing A technique used by 3D printers to print (create) three-dimensional objects. In this technique, manufacturing material is combined/joined/added/glued together to create a 3D object.

Add-on board A circuit board that provides additional capabilities for a computer. Also known as adapter card, expansion board, add-on card.

Address An identification, represented in the form of a name, label, or number, for designating a location in storage area.

Address bus A bus that carries the address of an I/O device to be accessed by the CPU.

Address register A local storage register that contains address of the next instruction to be executed.

Agile development methods See Lightweight SDLC models.

Algorithm A sequence of precise and unambiguous instructions for solving a problem in finite number of operations.

ALGOrithmic Language (ALGOL) A programming language that was designed for specifying algorithms, primarily of scientific nature. It was widely used in Europe.

Alpha testing After completion of integrated testing of software, it is released for further testing to a very select user group within or outside the organization. This group tests the software by using it as if they are its users. This testing is known as alpha testing.

Alpha version After completion of integrated testing of software, it is released for further testing to a very select user group within or outside the organization. This group tests the software by using it as if they are its users. This release version of software is known as alpha version.

Alphabetic Pertaining to a character set that contains letters A to Z in both lowercase and uppercase, and blank character.

Alphanumeric Pertaining to a character set that contains letters, digits, and other special characters such as comma, dollar sign, plus sign, etc.

American National Standards Institute (ANSI) A U.S. based national organization that establishes uniform standards in several fields.

American Standard Code for Information Interchange (ASCII) A standard coding system for computers. ASCII-7 is a 7-bit code, and its extended version, ASCII-8 is an 8-bit code.

Analog cellular A wireless communication technology that enables computing equipment, equipped with a modem, to transfer data over analog cellular telephone networks.

Analog computer A computer that operates on data, which is in the form of continuously variable physical quantities such as electrical current.

Analog transmission A communication system, which transmits analog signals and uses amplifiers at periodic intervals along communication lines between modems to amplify (strengthen) weak analog signals.

Android OS An open source operating system for mobile devices. It is owned by Google.

Animation A multimedia component that contains a set of images (called frames) displayed at a reasonable speed to create an effect of visual change or motion, similar to a movie film (video).

Anonymous ftp site A computer on the Internet that allows a user to log in with a username of anonymous and then a password, which is user's e-mail address.

Anti-spam technique An approach for automatic elimination of unwanted (spam) e-mails.

Antivirus utility A program used to cure a computer from virus infection.

Applets Segments of programs that act as mini-applications.

Application program A program included in an application software package.

Application programmer A programmer who prepares application software.

Application software A set of one or more programs designed to solve a specific problem or do a specific task such as pay calculation, processing of examination results, stores accounting and inventory control, etc.

Architecture Organization and interconnection of various components of a computer system.

Archiving Process of storing old data files on separate disks or tapes to make room for new files on hard disk of a computer.

Arithmetic Logic Unit (ALU) Unit of a computer that performs all mathematical and logical operations. It is one of the components of the central processing unit (CPU) of the computer.

Arithmetic operators Operators used for arithmetic operations in programs (such as x + y).

Array A data type that is a collection of elements in which all elements are of the same type. It has a fixed number of elements, called its length or size, given at the time of its declaration. All elements of an array are stored in contiguous memory locations.

Artificial intelligence A branch of computer science that deals with possessing, reasoning, learning, and thinking capabilities of computers that resemble those of human beings.

Aspect ratio Ratio of horizontal to vertical dimensions. It is 4:3 for both TV and computer monitors.

Assembler A computer program that translates an assembly language program to its machine language equivalent.

Assembly language A programming language that allows instructions and storage location to be represented by letters and symbols instead of numbers. Also known as symbolic language.

Assembly program A program written in an assembly language. Also known as symbolic program.

Assignment operators Operators used for assignment operations in programs (such as x = y).

Asymmetric cryptosystem Cryptosystem in which decryption key (Kd) is not equal to encryption key (Ke). Furthermore, it is computationally impractical to derive Kd from Ke. Hence, the system requires only Kd to be kept secret. Ke is made publicly known. Asymmetric

cryptosystems are also known as public-key cryptosystems.

Asynchronous communication Data transmission mode in which sender sends data at any convenient time and receiver accepts it.

Attacker See Intruder.

Audio A multimedia component that contains sound.

Audit logs Management technique in which a system maintains a record of all logins. That is, for each login, it records such information as time of login, duration of login, accessed objects, and types of accesses made.

Authentication Process by which a system verifies the identity of a user (person or program) before it permits the user to access a requested resource.

Authentication server (AS) A trusted entity that is shared by all communicating users of the system. It is used for authentication of communicating entities.

Authorization Once a system authenticates a user/process successfully, the system should next have ways to prohibit the user/process from accessing those resources/information that he/she/it is not authorized to access. This issue is called authorization. A system deals with this issue by using access control mechanisms.

Automated tape library A mass storage device that uses a set of magnetic tapes, tape drives, and a controller mounted in a single unit to function as a single large storage unit. Its typical use is for data archiving and as on-line data backup device for automated backup.

Automatic Teller Machine (ATM) An unattended machine used in banks that allows a customer to deposit or withdraw cash by using an electronic card without the need to interact with a bank employee.

Auxiliary memory A storage that supplements main memory of a computer. Often referred to as secondary storage, this section of a computer's memory is non-volatile and has low cost per bit stored, but its operating speed is generally far slower than that of primary storage.


B

Backbone Main trunk lines of a computer network that facilitate interconnection of other computers and computer networks on the same network.

Back-end computer A computer, which handles all I/O operations, relieving host computer of locating an I/O device and transferring data to/from it.

Background processing Automatic execution of lower-priority (background) computer programs, when higher-priority (foreground) programs are not using system resources.

Backup Alternate facilities of programs, data files, hardware equipment, etc., which are used in case the original one is destroyed, lost, or fails to operate.

Backup file Copy of a file created as a safety precaution against loss of data caused due to corruption, or inadvertent deletion of the original file.

Backward compatible A manufacturing strategy in which a manufacturer ensures that when a new CPU is developed, its instruction set includes all instructions in the instruction set of its predecessor CPU plus some new ones.

Band printers See Chain printers.

Bandwidth Range of frequencies available for data transmission. Wider the bandwidth of a communication system the more data it can transmit in a given period.

Bar codes Representation of alphanumeric data by a combination of adjacent vertical lines (called bars) by varying their width and spacing between them. A bar-code reader is a scanner used for reading (decoding) bar-coded data.

Base Total number of digits (symbols) available to represent numbers in a positional number system.

Basic Input Output System (BIOS) Programs stored in a ROM chip of a computer that checks the input and output devices when the system is started.

Basic Multilingual Plane (BMP) In Unicode, an area of codespace that contains code points for commonly used characters (about 64K code points), and space for about 6,300 unused code points for future expansion.

Batch processing Running of several computer programs one after another without the need of a human operator to run each program individually.

Baud Unit for measuring data transmission speed of a carrier. Generally, a baud is identical to bits per second (bps).

Beginners All-Purpose Symbolic Instruction Code (BASIC) An easy-to-learn, high-level, interactive programming language that is used frequently with personal computers, and in timesharing environments.

Beginning of Tape (BOT) marker A reflective metal foil, after unused length towards the beginning of tape, which indicates the beginning of usable tape to tape drive.

Bernoulli disk A magnetic disk-based storage medium in which a single hard disk platter is encased in a plastic cartridge.

Beta testing Use of beta version of software by a selected group of users for identifying errors that might have been overlooked during regular testing phase.

Beta version Version of software that is released to a selected group of users for testing before its actual release.

Bi-directional printing A printing method in which many dot-matrix printers print both ways - while the print head moves from left to right and while it moves from right to left, on return.

Binary A characteristic or property involving selection, choice, or condition, in which there are two possibilities.

Binary Coded Decimal (BCD) One of the early coding systems that is based on the idea of converting each digit of a decimal number into its binary equivalent, rather than converting the entire decimal value into a pure binary form. For example, the decimal number 42 is represented as 0100 0010 in 8-4-2-1 BCD notation.

Binary number system A number system with a base of two. It consists of two digits: 0 and 1.

Binary search tree A binary tree with following properties: (a) the data element of each node has key field and no two nodes have the same key value (all keys are distinct); (b) the keys of all nodes in the left subtree of the root have values less than the key in the root; (c) the keys of all nodes in the right subtree of the root have values greater than the key in the root; (d) each subtree

is itself a binary search tree.

Binary semaphore A semaphore whose value can range only between zero and one. It is suitable for synchronizing access to those mutually exclusive resources that have only one unit in a system.

Binary tree A special type of tree in which the out-degree of any node is less than or equal to two (zero, one, or two).

Bio databases In bio-informatics area, numerous genome sequencing projects going on all over the world, like Human Genome Project, have led to availability of millions of sequence data, which are stored in huge databases called bio databases.

Bioinformatics Deals with computer applications to solve those problems of biology that involve data emerging from within cells of living beings. Some of the main branches of bioinformatics are Genomics, Proteomics, Computer-Aided Drug Design, Data Mining of Bio Databases, Molecular Phylogenetics, Microarray Data Analysis, and Molecular Modeling of Biomolecules.

Biometric device A device used for user authentication that verifies some physical characteristic of a user such as the person's appearance, fingerprint, hand geometry, voice, or signature.

Bit Acronym for binary digit. It stands for one binary digit - 0 or 1.

Bitmap A method of displaying graphic images on a monitor by altering the color (or turning on or off) of individual dots (pixels) on the coordinate system.

Bits per second (bps) Unit of measurement of bandwidth (also called baud).

Bitwise operators Operators used for bit-level operations in programs (such as x & y, meaning x AND y).

Black-box testing Software testing method in which the tester treats the software as a black-box because he/she has no knowledge of its internal implementation. The tester does not have access to the internal data structures, algorithm, or the code of the software.

Block (1) A group of related items (records, characters, etc.) handled as a unit during input and output. (2) A section of program code treated as a unit.

Blocking factor Process of grouping two or more records together to form a block, and the number of records put together in each block is called blocking factor.

Bluetooth It is an industrial specification for short range wireless connectivity using globally unlicensed short range radio frequency. It provides a way to establish wireless connectivity between devices such as PCs, laptops, printers, mobile phones, and digital cameras for exchange of information between them.

Boolean algebra Algebra that deals with logical propositions, which are either true or false. It is suitable for use with binary number system, and is very useful in designing logic circuits that are used by the processors of computer systems.

Boolean function A mathematical function in Boolean algebra. For example, w = x + y × z.

Boolean variable A variable used in Boolean algebra. Its value can be true (1) or false (0).

Booting Process of automatic loading of operating system to primary storage, and readying a computer system for use.

Bottom-up approach A disciplined approach to system design or program design, in which the designer first selects base elements from available software libraries and/or existing applications, and builds the system by synthesizing base elements into subsystems, and subsystems into large subsystems, and so on until a complete top-level system is formed. Reusability of code is one of the main benefits of this approach.

Branch instruction An instruction that transfers program control to a non-consecutive instruction.

Branch statement See Control statement.

Bridge An internetworking tool that is used to connect networks that use the same communication protocols above data-link layer, but may not use the same protocols at physical and data-link layers.

Broadband channel Communication channels that are capable of data transfer rates of 1 million baud (bits/second) or more. They are used to handle high volumes of data, and for high-speed computer-to-computer communication, or for transmission of data to several different devices simultaneously.

Browser Software tool that provides several navigation facilities to help users do Internet surfing easily and effectively.

Browsing A method of passive attack in which an intruder attempts to read stored files, message packets passing by on a network, other processes' memory, and so on, without modifying any data.

Buffer Small storage area used to store information temporarily to compensate for the difference in rates of data flow between various units of a computer. For example, when data flows from I/O unit to CPU, it passes through a buffer.

Bug An error in a computer program.

Built-in functions Subprograms provided with a programming language so that a programmer can call them in a program to have them invoked automatically. Also known as intrinsic subprograms.

Bulletin Board System (BBS) The electronic counterpart of a wall-mounted bulletin board that allows users of a computer network to exchange information (by posting or reading messages) via a centralized database.

Bundled Inclusion of software with hardware of a computer system as part of the system's total price.

Bus A set of wires, which carries a group of bits in parallel and has an associated control scheme. It provides a communication path between two or more devices of a computer system. For example, the path between the memory and a peripheral device.

Bus mouse A mouse that requires a special electronic card, which provides a special port for connecting the mouse to a computer.

Byte A fixed number of adjacent bits that represent a particular character or symbol. Normally, a byte consists of eight bits.


C

C A programming language, having virtues of high-level programming languages (mainly machine independence) and efficiency of assembly languages.

C# It is read as C sharp. It is an object-oriented programming language developed by Anders Hejlsberg and released by Microsoft. It is part of Microsoft's .NET technology initiative.

C++ An object-oriented programming language that is an extension of C language.

Cache memory A small high speed memory that is used to increase speed processing by making current programs and data available to CPU at a rapid rate.

Call statement A program statement that transfers program control to a subroutine.

Canonical form A Boolean function whose terms contain all variables (or their complements). This is unreduced form of the function in minterm or maxterm form.

Carrier Any device that is used to transmit data from one location to another.

Cathode An oxide-coated metal in electron beam generator.

Cathode Ray Tube (CRT) An electronic tube with a TV like screen on which information may be displayed.

C-band transmission Use of 4 GHz band of frequencies for transmission and retransmission of microwave signals in a satellite communication system.

CD Read/Write (CD-RW) disk An optical storage that is very similar to a WORM disk with the exception that one can erase the previous content and write on it multiple times.

CD-Recordable (CD-R) disk See Write-Once, Read-Many (WORM) disk.

CD-ROM jukebox A mass storage device that uses a set of CD-ROM disks, disk drives, and a controller mounted in a single unit to function as a single large storage unit. Its typical use is for archiving read-only data.

Cell (1) Intersection of a row and column in a spreadsheet. A user enters text, number, or formula in a cell. (2) Area covered by an individual transceiver's signal in a wireless communication system.

Cellular Digital Packet Data (CDPD) A wireless communication technology in which digital data to be transferred is broken into small packets and inserted into analog cellular telephone networks.

Central Processing Unit (CPU) Control unit and arithmetic logic unit of a computer system together constitute its CPU. It is the brain of a computer system and carries out all calculations and comparisons performed by the computer. It is also responsible for activating and controlling operations of other units of the computer.

Certification Authority (CA) An organization that issues public key certificates to users. A public-key certificate (known as certificate in short) binds the public key of a user (generated by the CA) with his/her/its details such as name, address, e-mail, etc.

Chain printers Line printers that print by striking a set of hammers on an inked ribbon and paper placed against a metallic chain. All characters of the character set supported by the printer are embossed on the metallic chain.

Challenge-response A method in which a user proves his/her identity by responding correctly to challenge questions asked by the verifier.

Channel (1) A path for carrying signals between a source and destination. (2) A track on a magnetic tape, or a band on a magnetic drum.

Character addressable storage A storage in which each character has a unique location with its own address. Another name for it is variable-word-length storage.

Character coding schemes Standards for representing characters in computers. Character printer A printer with a print mechanism that prints one character at a time. Character set Valid characters allowed in a programming language.

Charge-Coupled Device (CCD) A completely electronic storage device fabricated on semiconductor chips. It stores data as pockets of charge in a semiconductor.

Chip A thin wafer of silicon on which integrated electronic components are deposited.

Chosen-plaintext attack Cryptosystem attack in which an intruder has access to ciphertext for any plaintext of his/her choice. The intruder tries to derive its decryption key by examining several ciphertexts for carefully thought plaintexts of his/her choice.

Ciphertext-only attack Cryptosystem attack in which an intruder intercepts ciphertext and derives its decryption key from the ciphertext. A system that cannot resist a ciphertext-only attack is totally insecure and useless.

Circuit switching The simplest method of data communication in which a dedicated physical path is established between a sender and receiver nodes through switching nodes of the network for the duration of data exchange.

Class In OOP, a class is a description of one or more similar objects.

Class variables In OOP, class variables are variables stored in a class whose values are shared by all instances (objects) of the class.

Client computer A computer that makes service requests to a server computer in a client-server computing environment. It runs client processes, which send service requests to the server.

Client-server computing A network of computers consisting of a server computer (such as file server, database server, print server, name server) and multiple client computers that share programs and data from the server.

Clinical information system A medical informatics application that deals with maintaining records of all types of clinical tests including radiology, pathology, and cardiology.

Clip art library A library of commonly used graphic images or objects that can be imported directly from the library and used in a multimedia application.

Clock cycles See System clock.

Clock speed Number of times the built-in electronic clock of a computer system pulses in one second.

Clone A hardware or software product that emulates an established product having good reputation in the market.

Cloud computing A new paradigm that delivers all types of computing and communication resources as services including hardware resources (such as compute power engine and storage), software resources (such as software libraries, applications, and programming tools), and communication resources (such as communication services and collaboration tools). It enables

users to access these resources on-demand using various types of access devices (such as PC, Workstation, Laptop, Smartphone, and Mobile phone). A user pays for only the type and amount of services he/she uses.

Cluster of Workstations (COWs) A system that consists of several standard workstations interconnected with a standard network such as Myrinet or Gigabit Ethernet.

Clustering Allows multiple servers to be treated as a single server so that the power of multiple servers can be combined to service a single request needing a powerful server, which no single server has the capability to handle alone.

Clustering technology A technology for making parallel computers by interconnecting powerful commodity workstations with a high-speed commodity switched network.

Coaxial cable A cable made of a single wire in center of an insulator wrapped in mesh. It is often used as a medium to transmit data between computers, and between computers and peripheral devices.

Code (1) The statements of a computer program. (2) A set of rules outlining the way to represent data (like BCD, EBCDIC, ASCII).

Code blocks In Unicode codespace, characters are grouped together by scripts in code blocks.

Code Division Multiple Access 2000 (CDMA 2000) The technology standard for 3G in America.

Code generator A very high-level programming language in which programmers specify the design specifications of a system (which tasks are to be performed), through an interactive dialog with the system. The computer then automatically generates (produces) the program source code for the system.

Code point A single number that is assigned by Unicode to each simple character.

Coding Process of writing computer instructions in a programming language. Cognitive graphics Area of computer graphics that deals with images (see Images). Collate To combine items from two or more sequenced files into a single one.

Collating sequence An ordering assigned to characters of a character set to be used for sequencing purposes. A commonly encountered collating sequence involves special characters, letters, and digits, in that order.

Collision A problem in a hashing algorithm, which occurs when the algorithm maps key values of two or more records to same storage address.

Color and graphics adapter card A hardware that enables interfacing a PC with graphics and/or color video monitor.

Combinational circuit A group of logic gates interconnected to form a logic circuit.

Command interpretation A module (known as command interpreter) of an operating system that provides a set of commands, which users use to give instructions to a computer for execution of their jobs.

Command interpreter The part of an operating system that serves as an interface for users to communicate with a computer via its operating system.

Command-line interface A textual user interface in which users give instructions to computer

by typing commands on keyboard.

Comment A statement written in a computer program for the purpose of documentation. Comments do not contain any program logic, and are ignored (not translated) by the language processor.

Commercial online service A method of connecting a computer to the Internet, in which a service organization offers a complete online environment and access to some or all of the Internet's services by using telephone service to connect a client's local call to the gateway computer of the service organization.

Common Business Oriented Language (COBOL) A high-level programming language developed for business data processing applications.

Common carriers Data-transmission-service providers are known as common carriers. Types of services, which common carriers typically offer are dial-up line, leased line, Integrated Services Digital Network (ISDN), and Value Added Network (VAN).

Communication protocol A set of rules and procedures established to interconnect and communicate between computers. It provides a method for orderly and efficient exchange of data by establishing rules for proper interpretation of controls and data transmitted as raw bits and bytes.

Communication security Safeguards against unauthorized tampering of information while it is being transmitted from one computer to another through communication channels.

Communications processor A processing unit that coordinates networks and data communications. Within a computer network, it ensures that data flows to and from different computer systems correctly and efficiently.

Communications satellite A microwave relay station precisely positioned 36000 kms. above the equator with an orbit speed that exactly matches the earth's rotation speed. It enables data transmission between any two randomly chosen points in a large area.

Communications software Software that enables transfer of data and programs from one computer system to another in a network of computers.

Compact Disk-Read-Only Memory (CD-ROM) A shiny, silver color metal disk of 5¼-inch (12 cm) diameter having a storage capacity of about 650 Megabytes. It is so called, because of its enormous storage capacity on a compact-size disk, and because it is a read-only storage medium.

Compile To translate a program written in a high-level language into its equivalent machine language program. Usually a single source statement yields more than one machine instruction.

Compiler A translator program (software package) that translates a high-level language program into its equivalent machine language program.

Complement For a number having n digits, complement is defined as the difference between the number and the base raised to the nth power minus one. For example, complement of 3710 = 102

- 1 - 37 = 6210. In Boolean algebra, complement of a variable is the reverse (NOT) of its value. For example, complement of A is `A.

Complementary subtraction Additive method of subtraction.

Complete tree A tree that has every level filled in before adding a node to the next level, and one that has the nodes in a given level filled in from left to right, with no breaks.

Completely connected network A network in which there is a separate physical link for connecting each node to any other node.

Complex Instruction Set Computer (CISC) Architecture An architecture of CPUs with large instruction set, variable-length instructions, and a variety of addressing modes.

Component A self-contained piece of software that has a well-defined interface and can be developed, delivered, installed, and run independently.

Component video signal A video signal that separates video into luminance Y and chrominance

C streams.

Composite character In Unicode, a character, like “ü” or “ n ”, which is often used in Latin.

Composite video signal A video signal that combines luminance Y and chrominance C into a single stream, which users can store or transmit.

Compound statement A statement to perform complex operations combining null, simple, and other compound statements. Also called statement block.

Computer An electronic device that can automatically accept and store input data, process them, and produce output results by interpreting and executing programmed instructions.

Computer Aided Design (CAD) A multimedia application of generative graphics that enables use of computers to automate design and drafting operations. CAD packages are available for designing a variety of things - cars, aircrafts, ships, buildings, printed circuit boards, etc.

Computer Aided Instruction (CAI) A general term that refers to a learning situation in which the student interacts with (and is guided by) a computer through a course of study aimed at achieving certain instructional goals. Also called CAE (Computer Aided Education).

Computer Aided Manufacturing (CAM) Use of computers to automate manufacturing operations.

Computer Aided Software Engineering (CASE) tools A set of software tools that provide a wide range of features for creation of better and more reliable software products within less time and with less effort.

Computer animation Area of computer science that deals with generating, sequencing, and displaying (at a specified rate) of a set of images (called frames) to create an effect of visual change or motion, similar to a movie film (video). Animation is commonly used in those instances where video shooting is not possible, or animation can better illustrate the concept than video.

Computer audio Area of computer science that deals with synthesizing, recording, and playback of audio or sound with a computer.

Computer codes See Character coding schemes.

Computer Emergency Response Team (CERT) It is established by a country's government with the purpose to become the nation's most trusted referral agency for responding to computer security incidents as and when they occur.

Computer graphics Area of computer science that deals with generation, representation, manipulation, and display of pictures (line drawings and images) with a computer.

Computer literacy Having sufficient computer knowledge to work and live in a computerized society.

Computer network A network that connects multiple computers in a manner to enable meaningful transmission and exchange of data among them with the main objective of sharing of information, resources (both hardware and software), and processing load among the connected computers.

Computer Output Microfilm (COM) An output device that records computer output on a sheet or roll of microfilm as microscopic filmed images. It uses a microfilm recorder for recording output information on a microfiche, and a microfilm reader to view the information recorded on a microfiche.

Computer Supported Cooperative Working (CSCW) An application in which multiple members working on a single project and located at distant locations cooperate with each other using a network of computers.

Computer system Various components (input and output devices, storage, CPU) of a computer integrated together to perform the steps called for in a program being executed.

Computer video Area of computer science that deals with recording and display of a sequence of images at a reasonable speed to create an impression of movement.

Computer-aided drug design A branch of bioinformatics that deals with using computational techniques to minimize time and effort for searching drug molecules.

Computer-Based Training (CBT) Using computer technologies for training and education.

Concentrator An intelligent multiplexer that performs the same function as a multiplexer, but has the ability to actually reduce the number of signals.

Conditional branch statement A branch statement that causes alteration in sequential flow of execution of a program with some condition test (such as if statement).

COnference on DAta SYstems Languages (CODASYL) A committee of computer users, manufacturers, and the United States government, which helps to establish programming standards for various programming languages.

Confinement problem The problem of ensuring that it is impossible for a potential accomplice to leak any information to the outside world.

Connection-establishment server A firewall that establishes only permitted TCP connections between a user on a trusted network and a user on an untrusted network.

Console In a mainframe system, the terminal that is directly connected to the host (main computer), and is mainly used by the system administrators to monitor the health of the system, or to perform some system administration activities.

Constant A value, written into a program instruction, which does not change during execution of the program.

Contiguous allocation A disk space allocation method, in which each file occupies a set of contiguous blocks on a disk.

Control bus A bus that carries commands such as START, READ, WRITE, REWIND TAPE, etc., from CPU to I/O devices.

Control characters Special characters for controlling such activities as printer vertical spacing, movement of cursor on terminal screen, etc.

Control program An operating system program that controls the operations and management of

resources of a computer system. Its major functions are job scheduling, input/output scheduling, and program monitoring.

Control statement See Control structures.

Control structures Decision statements in a program that control the flow of execution of the program based on some condition or abruptly. Also called control statements or branch statements.

Control unit The unit of a computer system that manages and coordinates the operations of all its other components. It is one of the components of the central processing unit (CPU) of a computer.

Controlling Directing the manner and sequence in which inputting, storing, processing, and outputting operations are performed.

Copying utility A file utility for copying a file either from one unit of a storage device to another similar unit (such as from one tape reel to another), or from one storage medium to another (such as from tape to hard disk, or from CD-ROM to hard disk). It is also known as peripheral interchange program (PIP).

Cordless mouse A mouse that is not connected to a computer and it communicates with the computer with the help of a special controller that operates by transmitting a low-intensity radio or infrared signal.
