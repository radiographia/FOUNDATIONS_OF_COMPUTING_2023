Major Elements of Internet Search Engines‌

The three major elements of an Internet search engine are as follows:

1. Search request interface enables users to provide description of the desired information to the search engine. Depending on the complexity of this element, the search engine may allow specifications of simple keywords and phrases, combination of keywords and phrases using Boolean operators and exclusion/inclusion operators, and title and URL

limiters.

2. Information discoverer discovers information from the WWW and creates a database for the search engine. The search engine uses the database to locate useful information during the search process. Information discoverer accrues this information in two-ways: (a) in manual method, authors provide information about their websites to the search engine; (b) in automatic method, information discoverer collects the information using programs, such as web crawlers, spiders, robots. These programs periodically roam on the Internet, and store links of information about each page they visit along with their associated hyperlinks. Information discoverer then creates an index from the results of automated search. If a web page owner updates his/her/its web page, the spider finds these changes and updates its index.

3. Presenter of search results returns search results, ranking them in an order it believes, is most relevant to the search. Search engines often list search results in accordance with a relevance score. Relevance scores reflect the number of times a search term appears in a web page. Some search engines also allow users to control relevance score by giving different weights to each search word.
