BOOLEAN FUNCTIONS‌

A Boolean function is an expression formed with binary variables, Boolean operators (OR, AND, and NOT), parentheses, and equal to sign. This Boolean function is in the form of an algebraic expression. A truth table with 2n rows can also represent a Boolean function, where n is the number of literals (binary variables) in the function. The combinations of 0s and 1s for the rows of this table are obtained from the binary numbers by counting from 0 to 2n - 1.
