Word-processing Package‌‌

The term word-processing describes the use of hardware and software to create, edit, view, format, store, retrieve, and print documents (written material such as letters, reports, books, etc.). A word-processing package enables us to do all these on a computer system.

Some commonly supported features found in modern word-processing packages are:

1. Entering text

2. Editing text

3. Formatting page style

4. Formatting text

5. Entering mathematical symbols

6. Displaying documents

7. Saving, retrieving and deleting documents

8. Printing documents

9. Importing text, graphics and images

10. Searching and replacing text string

11. Checking spelling

12. Checking grammar and style

13. Using thesaurus

14. Creating table-of-contents and index

15. Using mail-merge
