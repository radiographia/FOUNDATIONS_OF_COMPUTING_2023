SSL Alert Protocol‌

It is a two-byte message that conveys SSL-related alerts to the peer entity (see Figure 15.23). The first byte indicates alert level - warning (value = 1) or fatal (value = 2), and conveys the severity of alert. SSL terminates the connection immediately in case of a fatal alert message. It allows continuation of other connections on the same session, but disallows establishment of any new connection on this session. The second byte contains a code indicating the specific alert.
