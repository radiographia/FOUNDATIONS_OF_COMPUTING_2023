Uses of Magnetic Disks‌

We use magnetic disks for the following purposes:

1. For random data processing applications.

2. As a shared, on-line secondary storage device. We often use Winchester disks and disk packs for this purpose.

3. As a backup device for off-line data storage. We often use zip disks and disk packs for this purpose.

4. Archiving of occasionally used data. We often use zip disks and disk packs for this purpose.

5. Transferring of data and programs between two unlinked computers. We often use zip disks for this purpose.
