Text‌

Text information contains alphanumeric and special characters. Most modern computers have text processing capability to deal with text information.


Hardware Requirements for Text

Normally, modern computers use the following hardware devices for text processing capability:

1. Keyboards to input text data.

2. Electronic writing pads for direct input of handwritten text.

3. OCRs (Optical Character Recognizers) for direct input of printed text.

4. Monitors (terminal screens) to display text information.

5. Printers to output text in hard copy form.


Software Requirements for Text

The following text-processing capabilities are desirable in a multimedia computer system for better presentation and use of textual information:

1. Text editing. Text editors and word processing packages enable generation, editing, and planning the layout of a text document.

2. Text style. Presentation of textual information becomes more effective by using text of various sizes, fonts, and styles (bold, italics, shadow, etc.). For example, newspapers use a good combination of all these styles to highlight different news differently, depending on their degree of importance. Similarly, a book like this one uses a combination of various text styles to put more emphasis on certain words or phrases than remaining text.

3. Text searching. This feature greatly enhances the ease of use of text information. It enables a user to quickly search the location(s) of a word or phrase in a text document, which may be of several pages.

4. Hypertext. This feature provides great flexibility of structuring and accessing computer- based text documents. It generalizes the concepts of “footnote” and “cross reference” of traditional way of organizing text documents and enables users to obtain the desired information by clicking on a “hotword” (a word or phrase linked to another document) within a document. Hence, to obtain more information about a hotword, the user selects the hotword (usually with a mouse click on it), and the system automatically opens a window with additional text explaining that word or phrase. This type of information structuring allows non-sequential navigation within the information space according to individual needs. A system distinguishes hotwords from other text in a document by displaying them in different color or font, or by changing the shape of the mouse pointer (such as a hand instead of an arrow) when a user moves the mouse over a hotword. This simple navigation aid makes it easier for a user to identify hotwords in a document.

5. Text importing and exporting. This feature enables a user to import already existing text (partly or fully) into a new document at a desired location, instead of retyping the entire text again. If an imported file is large, it will result in enormous saving of time and effort required to prepare the new document. Similarly, text-exporting feature is useful in allowing other packages to use an existing document (partly or fully).
