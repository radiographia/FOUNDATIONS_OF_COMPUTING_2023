Octal Number System‌

In octal number system, the base is 8. Hence, there are only eight symbols or digits: 0, 1, 2, 3, 4, 5, 6, and 7 (8 and 9 do not exist in this system). The largest single digit is 7 (one less than the base 8). Each position in an octal number represents a power of the base (8). Therefore, decimal equivalent of octal number 2057 (written as 20578) is:

(2 x 83) + (0 x 82) + (5 x 81) + (7 x 80) = 1024 + 0 + 40 + 7 = 1071

Hence, 20578 = 107110

Since there are only 8 digits in octal number system, 3 bits (2 3 = 8) are sufficient to represent any octal number in binary (see Figure 2.1).
