Media Center PC Features and Functionalities‌

A media center PC works as a PC, TV, music system, radio, playstation/gameboy, and DVD/VCD player and recorder. In addition to keyboard, mouse, and display screen, it has a TV tuner and a remote control (see Figure 14.18). Of course, it also has a powerful graphics processor based display card, large memory, fast hard disk with large storage space, and a digital surround soundboard. It has an easy-to-read and visually attractive graphical user interface (GUI), which is easy to operate even for a person who has no experience of working on a PC. It is as easy to use as a TV.


Figure 14.18. A media center PC.


Apart from other facilities, a media center PC normally provides the following functionalities:

1. It serves as a PC.

2. It serves as a TV.

3. It enables not only viewing of TV programs, but also allows a user to pause and record live TV programs. For example, if a telephone call or doorbell interrupts a user while he/she is viewing a TV program, the user can pause the program, attend to the call, and can start viewing the program again without missing out a single dialogue or scene. In such a situation, the user can also fast forward through the advertisements to catch up with the live TV program.

4. It provides Electronic Programming Guide (EPG) feature that tells a user what is playing on what channel and when during the next few days. With EPG feature, a user can search for his/her favourite program(s), and in case the user is not around at that time, he/she can set a recording so that the system records the program for the user to view it at his/her own convenience. This feature also enables the user to record entire series of his/her favourite program (such as all episodes during the week of a serial drama). This feature is dependent on area and subscribed services.

5. It allows a user to chat with his/her friends while watching TV on the same screen.

6. It serves as a music system with enhanced features. A user can create his/her own digital music library of songs in the system and can play favourite songs from the library without the hassle of changing CDs. As the system has a CD drive, a user can also play songs from a CD or record/copy songs on a CD.

7. It serves as a radio. It enables not only listening to radio programs, but also allows a user

to pause and record live radio programs and listen to them at his/her own convenience.

8. It serves as a digital photo album. It serves not only as storage of digital pictures, but also allows a user to index, edit, and view them with added features (zoom, rotate, auto advance, etc.).

9. It serves as a digital video library. A user can maintain his/her own library of digital video (such as friend's wedding video, excursion trip video, family video, lecture video, etc.) and can enjoy them at his/her own convenience.

10. It serves as multimedia data editing and enhancement tool. A user can use this feature to create customized multimedia content for future use. For example, a user can create a customized music by mixing other already available music files. Similarly, a user can create a customized video or animation by combining available clips and photographs.

11. It serves as DVD/VCD player and recorder. A user inserts a DVD/VCD in DVD/VCD drive of the system to play/record multimedia data on it.

12. It serves as a videogame gadget such as playstation/gameboy. It enables children to play computer games. It also enables them to play Internet games with other users over the network.

Any PC with good graphics display card, multimedia components, and a TV tuner card can serve as a media center PC by installing media center applications on it. Some operating systems provide features of media centre applications integrated into OS itself. Such operating systems are called media center OS.
